 How to play
----------------------------------------------------------------------

*Key assign*

x:Light Punch
y:Hard Punch
z:Emergency roll
a:Light Kick
b:Hard Kick
c:Strike attack
start:Taunt

*System*

KOF2001 mode
 -Select character without start button at character select screen.

KOF'98 mode
 -Select character with start button at character select screen.

Emergency roll
 -x+a/z

Strike attack
 -y+b/c

Taunt
 -start

Guard Cancel Emergency roll
 -(during gurding)x+a/z
 -Use power level at 1000

Guard Cancel Strike attack
 -(during gurding)y+b/c
 -Use power level at 1000

*Special Moves*

Tani Otosi
 -(near opponent)F+y

Tomoe Nage
 -(near opponent)F+c

Hyoutyu Wari
 -F+x

Jodan Uke
 -F+a
 -Can use only KOF2001 mode

Gedan Uke
 -DF+a
 -Can use only KOF2001 mode

*Super Moves*

Koou Ken
 -D,DF,F+x/y

Kohou
 -F,D,DF+x/y

Hien Shippukyaku
 -F,DF,D,DB,B+a/b

Zanretsuken
 -F,B,F+x/y

Mouko Raijinsetu
 -D,DF,F+a/b

Kohou Shippuuken
 -D,DB,B+x/y
 -Can use only KOF2001 mode

Kyokugenryu Renbuken
 -(near opponent)F,DF,D,DB,B+x/y

Mouko Raijingo
 -D,DB,B+x/y
 -Can use only KOF'98 mode

*Hyper Moves*

Hao Shokoken
 -F,B,DB,D,DF,F+x/y
 -Use power level at 1000

Hao Shokoken(MAX)
 -F,B,DB,D,DF,F+x/y
 -Use power level at 2000

Tenti Haouken
 -D,DF,F,DF,D,DB,B+x/y
 -Use power level at 1000

Tenti Haouken(MAX)
 -D,DF,F,DF,D,DB,B+x/y
 -Use power level at 2000

Tenti Haouken
 -D,DF,F,D,DF,F+x/y
 -Use power level at 1000

Tenti Haouken(MAX)
 -D,DF,F,D,DF,F+x/y
 -Use power level at 2000

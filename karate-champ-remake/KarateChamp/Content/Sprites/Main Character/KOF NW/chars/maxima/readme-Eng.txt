  Controls
  --------

    Basic Controls
    --------------
      D - down
      F - forward
      U - up
      B - back
      x - button X (weak punch)
      y - button Y (strong punch)
      z - button Z (armor mode)
      a - button A (weak kick)
      b - button B (strong kick)
      c - button C (counter mode)

    Standard Movement
    -----------------
      U              - Jump
      UF             - Jump forwards
      F              - Walk forwards
      DF             - Crouch
      D              - Crouch
      DB             - Crouch or low guard
      B              - Walk backwards or high guard
      UB             - Jump back
      F, F           - Forward dash
      B, B           - Back dash
      a+x            - Recovery (when falling)
       
    Basic Standing Attacks
    ----------------------
      x              - W. Punch
      y              - S. Punch
      a              - W. Kick
      b              - S. Kick
      b+y            - Knockdown attack
      F + a         - Neck Cracker
      F + x         - Mongorian 

    Basic Crouching Attacks
    -----------------------
      x             - Crouch W. Punch
      y             - Crouch S. Punch
      a             - Crouch W. Kick
      b             - Crouch S. Kick
      FD + y        - Crouch M-9 Maxima missile 

    Basic Air Attacks
    -----------------
      x             - Jump W. Punch
      y             - Jump S. Punch
      a             - Jump W. Kick
      b             - Jump S. Kick
      b+y           - Knockdown attack

    Throws
    ------
      F/B + y (when close) - Dynamite Drop
      F/B + b (when close) - Chocking Vice

    Special Attacks
    ---------------
      D, DF, F + x            - System-1 : Maxima Scramble
     ¦¦ D, DF, F + x          - Double Bomber
       ¦¦ D, DF, F + x        - Bulldog Press
           D, DF, F + y        - Maxima gallows
   
      D, DF, F + y            - System-2 : Maxima Scramble
     ¦¦ D, DF, F + y         - Skull Crash
       ¦¦ F + a or b          - 1000t. Press

      F, DF, D, DB, B + a or b - System-3 : Maxima Lift
     ¦¦ D, DF, F + a or b        - 1000t. Press

      D, DB, B + x or y           - M-4 Vaper Cannon

      D, F, DF + a or b           - M-19 britz cannon

      B, DB, D, DF, F + a or b (When close) - M-11 Dangerous Arch

      a+y                             - Max Mode activation            


    Hyper Attacks
    -------------
      D, DB, B, D, DB, B + a or b                           - Maxima Revenger
      D, DB, B, D, DB, B + c (2 power stock)          - Maxima Revenger(MAX)
      
      D, DF, F, D, DF, F + a or b                            - Britz Revenger(Original)
      D, DF, F, D, DF, F + c (2 power stock)           - Britz Revenger(MAX)
 
      D, DF, F, DF, D, DB, B + a or b                      - Bunker Buster
      D, DF, F, DF, D, DB, B + c (2 power stock)     - Bunker Buster(MAX)

      D, DF, F, DF, D, DB, B + x or y                      - Final Maxima Beam
